\section{Stakeholder Analysis}
\label{section:stakeholder-analysis}

\textbf{Traffic Police Officers} Traffic police officers need a system that automatically detects helmet violations and captures license plate numbers so that they can review violation records without manually watching CCTV footage.

\textbf{City and Municipal Authorities} City officials need access to violation data by location and time so that they can identify high-risk areas and allocate enforcement resources effectively.

\textbf{Highway and Road Departments} Road departments need automated monitoring on dispersed roads where maintaining a permanent officer presence is impractical.

\textbf{System Administrators} Technical staff need to configure cameras and maintain the system so that HelmTrack operates reliably over time.

\section{User Stories and Use Cases}
\label{section:user-stories-use-cases}

\subsection{Traffic Police Officers}

\begin{enumerate}[leftmargin=80pt]
    \item As a traffic police officer, I want to view a list of detected helmet violations so that I can review evidence and issue fines without manually watching CCTV footage.
    \item As a traffic police officer, I want to see the license plate number and captured image for each violation so that I can identify the offending vehicle.
    \item As a traffic police officer, I want to filter violations by date and location so that I can manage records more efficiently.
\end{enumerate}

\subsection{City and Municipal Authorities}

\begin{enumerate}[leftmargin=80pt]
    \item As a city authority, I want to view violation statistics by location and time so that I can identify high-risk areas and allocate enforcement resources effectively.
    \item As a city authority, I want to export violation reports so that I can use the data to support road safety policy decisions.
\end{enumerate}

\subsection{Highway and Road Departments}

\begin{enumerate}[leftmargin=80pt]
    \item As a road department officer, I want the system to monitor remote roads automatically so that helmet law enforcement is not limited to manned checkpoints.
\end{enumerate}
